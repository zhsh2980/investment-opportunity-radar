# Triage 标签

skill 用五个标准 triage 角色描述 issue 状态。本文件把这些角色映射到本仓库实际使用的标签字符串（写在 issue 文件的 `Status:` 行）。

| skill 中的角色 | 本仓库标签 | 含义 |
| --- | --- | --- |
| `needs-triage` | `待评估` | 需要维护者评估这个 issue |
| `needs-info` | `待补充` | 等报告人补充信息 |
| `ready-for-agent` | `AI可做` | 已完整说明，AI agent 可无人值守领取 |
| `ready-for-human` | `需人工` | 需要人工实施 |
| `wontfix` | `不做` | 不会处理 |

当 skill 提到某个角色（如「打上 AFK-ready 标签」）时，使用上表对应的中文标签字符串。
